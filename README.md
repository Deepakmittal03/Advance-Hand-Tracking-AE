## 🚀 Overview.

**Neon Aura AR** is a futuristic browser-based Augmented Reality (AR) experience that uses real-time hand tracking to create immersive visual and audio interactions.

This project leverages **MediaPipe Hands** and modern web technologies to transform hand gestures into dynamic neon effects, particles, and interactive animations — all running directly in the browser.

---

## ✨ Features

* 🖐️ Real-time Hand Tracking
* ⚡ Gesture Detection (Pinch, Open Hand, Fist)
* 🌈 Dynamic Neon Visual Effects
* 🎨 Multiple Themes (Rainbow, Cyberpunk, Lava, Ocean, Galaxy)
* 🔊 Interactive Audio Feedback
* 🧠 Physics-Based Particle System
* 🎥 Webcam-Based AR Experience
* 📊 Live Performance Stats (FPS, Hands, Gestures)

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **AR / ML:** MediaPipe Hands
* **Graphics:** Canvas API
* **Audio:** Web Audio API

---

## 🧩 How It Works

1. Webcam captures real-time video
2. MediaPipe detects hand landmarks
3. Gesture logic processes finger positions
4. Canvas renders interactive neon effects
5. Audio reacts dynamically to gestures
